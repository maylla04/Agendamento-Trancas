# Maylla Braids 💆‍♀️✂️

Aplicativo de agendamento para serviços de tranças.
Feito por Maylla Rieny Oliveira Nascimento

## Objetivo
Permite registrar agendamentos de clientes com nome, telefone, data, horário, tipo de trança e foto do cabelo.

## Funcionalidades
- Cadastrar novos agendamentos com foto
- Listar agendamentos salvos
- Visualizar detalhes e excluir agendamentos
- Armazenamento local com SQLite
- Acesso à câmera do dispositivo

## Tecnologias
- React Native com Expo
- TypeScript
- SQLite (expo-sqlite)
- React Navigation
- expo-image-picker
